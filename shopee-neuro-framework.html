medium effort margin สูง คู่แข่งน้อย

เร่งด่วน lever ใหญ่

Neuroscience of Buying — สมองลูกค้าทำงานยังไง

🧠
3 สมองที่ต้องชนะ (Triune Brain Model)
Reptilian Brain (สมองสัตว์เลื้อยคลาน) ตัดสินใจก่อน

ทำงานก่อนเหตุผล — ตอบสนองต่อ ความอยู่รอด / ความสวยงาม / สิ่งที่โดดเด่น
→ ชนะด้วย: รูปแรกต้องตีตาใน 0.3 วินาที, background ขาว/น้ำตาล, texture ไม้ชัด

Limbic System (สมองอารมณ์) สร้าง desire

ควบคุม emotion และ memory — ซื้อเมื่อ รู้สึกเชื่อมโยง หรือจินตนาการว่าตัวเองมีของนั้นแล้ว
→ ชนะด้วย: lifestyle photo (ของในบ้านจริง), video unboxing, คำพูดที่แตะ "บ้านอุ่น / ของดีส่งต่อรุ่น"

Neocortex (สมองเหตุผล) justify การซื้อ

เหตุผลที่คนใช้ อธิบายว่าทำไมถึงซื้อ หลังจากอยากได้แล้ว
→ ชนะด้วย: specs ชัด, comparison table ไม้สักvsMDF, คุ้มค่า long-term, รับประกัน

Dopamine Triggers — กลไก 5 ตัวที่ทำให้กดซื้อ

⚡
Scarcity + FOMO (กลัวพลาด)

Dopamine พุ่งเมื่อเห็นว่า "เหลือน้อย" หรือ "หมดแล้ว"
ใช้: "เหลือ 3 ชิ้นสุดท้าย" / "รับออเดอร์รอบหน้าอีก 2 อาทิตย์" / จำกัด batch custom

⚡
Social Proof (คนอื่นเขาเลือก)

Mirror neuron ทำให้สมองลอกพฤติกรรมคนอื่น — review + รูป real ทรงพลังกว่าคำโฆษณา 10 เท่า
ใช้: ขอรูป/คลิปหลังส่งสินค้า, highlight review ที่พูดถึงคุณภาพไม้

⚡
Anchoring (ยึดราคา)

สมองตัดสิน "แพง/ถูก" จาก reference แรกที่เห็น
ใช้: มี SKU ราคาสูงไว้ anchor เช่น set premium 3,500 → ทำให้ 1,200 ดูสมเหตุสมผล

⚡
Loss Aversion (กลัวสูญเสีย)

สมองรู้สึกเสียใจจากการ "สูญเสีย" มากกว่าดีใจจากการ "ได้รับ" ถึง 2.5 เท่า
ใช้: "ถ้าไม่ใช้ไม้สักแท้ ต้องเปลี่ยนใหม่ใน 5 ปี" / คำนวณต้นทุนระยะยาวให้เห็น

⚡
Identity Signaling (แสดงตัวตน)

คนซื้อของที่บอกว่า "ฉันเป็นคนแบบนี้" — ไม้สักแท้บอกว่า "ฉันเลือกของดี ไม่เอาของปลอม"
ใช้: positioning "สำหรับคนที่ไม่ประนีประนอมเรื่องคุณภาพ" / "ส่งต่อรุ่นลูกได้"

Quick Wins — ทำได้เลย impact ทันที

1
ส่งตรงเวลา 100% — ก่อนทำอย่างอื่น
Ship On Time penalty จาก Shopee คือตัวที่กด ranking ลงแรงที่สุด ถ้าแก้จุดนี้ได้ Ranking จะค่อยๆ ขึ้นเองโดยไม่ต้องเพิ่ม ads
2
Rewrite title + description ทุก SKU ด้วย keyword
ใช้เวลา 1-2 ชั่วโมง แต่ผล organic ranking ดีขึ้นใน 1-2 สัปดาห์ ไม่มีต้นทุนเพิ่ม
3
Review Campaign — ขอรูปหลังส่งสินค้า
ส่ง message หลังส่งของ 3 วัน ขอรูปแลก coupon 30-50 บาท — 10 review แรกเปลี่ยน algorithm ได้ชัดเจน
4
รูปแรก — A/B Test ด้วย Shopee Boost
ลอง 2 รูปหน้า สัปดาห์ละรูป ดู CTR — รูปที่ชนะคือรูปที่ algorithm จะ push ต่อ
5
เปิด Shopee Ads แบบ Smart — ไม่ใช่ Auto
ตั้ง Manual keyword bid สำหรับ keyword ที่ conversion rate สูง แทนที่จะให้ Auto burn งบกับ keyword กว้างๆ

รูปสินค้า — โครงสร้าง 5 ภาพที่ต้องมี

1
Hero Shot — ตีตาใน 0.3 วินาที
พื้นหลังขาว/น้ำตาลอ่อน, ไม้อยู่ตรงกลาง, เห็น texture ชัด, แสงดี ไม่มีเงา — นี่คือรูปที่ algorithm ใช้วัด CTR
2
Lifestyle Shot — จินตนาการในบ้านจริง
ถ่ายในบ้าน/มุมจริง มีของ prop เช่น ต้นไม้เล็ก/หนังสือ/กาแฟ — trigger limbic brain ให้ลูกค้า "เห็นตัวเอง" ใช้สินค้า
3
Detail Shot — texture + คุณภาพ
ใกล้มาก เห็น grain ไม้, รอยต่อ, ผิวเรียบ — สร้างความเชื่อมั่นและ justify ราคาที่สูงกว่า MDF
4
Size Reference — วัดเทียบให้เห็น
ถ่ายเทียบกับคน, เมตรไม้, หรือ infographic ขนาด — ลดความกังวลเรื่อง "ใหญ่เกินไปไหม" ก่อนซื้อ
5
Social Proof Shot — รูปลูกค้าจริง
รูปลูกค้าติดตั้งแล้ว + quote สั้นๆ — mirror neuron ทำให้คนอื่น "อยากเป็นแบบนั้นบ้าง"

Copy Psychology — คำที่สมองซื้อ

✍️
Words ที่ trigger dopamine
ไม้สักแท้ 100% ส่งต่อรุ่นลูกได้ ผลิตโดยช่างฝีมือ ทนทาน 20+ ปี ไม่มีสารเคมีอันตราย ขนาด custom ได้ จากป่าที่ถูกกฎหมาย Made in Thailand ซื้อครั้งเดียว ไม่ต้องเปลี่ยน
🚫
คำที่ต้องหลีกเลี่ยง
ราคาถูก พร้อมส่ง ของดีราคาไม่แพง ลดราคา XX%

คำเหล่านี้ทำลาย premium positioning และดึงลูกค้าที่ sensitive เรื่องราคา ซึ่งมักเป็น segment ที่ margin ต่ำที่สุด

90-Day Battle Plan — ทำน้อย ได้มาก

WEEK 1-2 — Foundation Fix
แก้ปัญหาพื้นฐานก่อนยิง ads
✓ Audit ทุก SKU: title, keyword, รูปแรก
✓ ตั้งระบบส่งของให้ On Time 100%
✓ ตอบ chat ภายใน 1 ชั่วโมง (ตั้ง auto-reply)
✓ ระบุ Top 3 SKU ที่จะโฟกัสก่อน
WEEK 3-4 — Content Attack
ยิง content + สะสม social proof
✓ ถ่ายรูปใหม่ Top 3 SKU ครบ 5 แบบ
✓ Rewrite description ด้วย framework Neuro-Copy
✓ เริ่ม review campaign (ขอรูปหลังส่ง 3 วัน)
✓ เปิด Shopee Ads Manual สำหรับ Top 3 SKU
WEEK 5-8 — Scale Winner
ดู data แล้ว double down
✓ SKU ไหน CTR ดี → เพิ่ม ads budget
✓ Keyword ไหน convert → bid เพิ่ม
✓ เพิ่ม lifestyle content ใน description (video)
✓ เริ่ม bundle offer สำหรับ upsell
WEEK 9-12 — Top of Mind
สร้าง brand moat ที่คู่แข่งลอกไม่ได้
✓ สร้าง Story ร้าน (about us + origin story)
✓ เปิด Flash Sale เฉพาะ follower ร้าน
✓ ดัน Affiliate ให้ช่วย push traffic
✓ วัดผล: ranking, conversion, revenue vs เดือน 1
